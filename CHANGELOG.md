# Changelog

## [Unreleased] - 2026-05-16

### 新增
- 新增圈子入口和服务端接入：App 可读取圈子信息流、发布图片作品、申请传输票据，并将圈子图片传输至土豆片设备。
- 新增圈子注册流程：注册时选择头像、填写用户名，并通过当前连接设备和系统通行密钥完成身份绑定。
- 新增圈子会话保存：注册成功后保存服务端返回的访问令牌和用户资料，用于后续发帖、申请传输票据和记录传输事件。

### 优化
- 参照注册页设计稿重构圈子注册 UI，补齐标题视觉、步骤条、设备卡、头像选择、用户名输入、通行密钥说明和主操作按钮的移动端交互状态。

### 说明
- 圈子信息流当前只展示作品元数据，不在 App 内直接查看图片；图片只在传输前通过短时票据下载到内存。
- 圈子 API 默认地址已指向内网服务器 `http://192.168.100.112:3000`，方便真机和局域网调试。
- 头像对象上传和老用户登录 UI 仍待补齐，当前客户端优先闭环新用户注册和发帖传输链路。
- 圈子服务端为独立仓库 `PotatoCircleBackend`，接口文档位于该仓库的 `docs/API.md`。

## [Unreleased] - 2026-05-13

### 优化
- 增强图像传输 `原始算法 4` 的真机显示效果：传输前提升饱和度、对比度并压暗中间调，使画面更接近小程序的鲜艳、深色表现。
- `原始算法 5` 同步使用相同增强，`原始算法 2 / 3` 使用较轻增强，`原始算法 1` 黑白逻辑保持不变。
- 将 6 色调色板中的绿色调整为纯绿，提升颜色识别和显示鲜艳度。
- 修复天气看板 AQI 三位数、污染等级徽标和污染物说明在固定模板内被裁切的问题，空间不足时优先省略“空气质量”前缀。
