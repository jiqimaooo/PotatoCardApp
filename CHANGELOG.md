# Changelog
## [Unreleased] - 2026-05-19

### 新增
- 圈子漂流瓶与普通信息流完成数据隔离：普通帖子只进入圈子信息流，漂流瓶只通过打捞接口随机返回。
- 漂流瓶打捞次数改由服务端统计，并支持在管理后台配置每日次数上限。

### 优化
- App 漂流瓶页改为每次打捞请求服务端随机内容，不再从普通圈子帖子缓存中随机取图。
- 拆分 Xcode 签名配置：共享项目只保留变量引用，个人 Team ID 和 Bundle ID 放到本地 `Signing.local.xcconfig`，避免多人打包时反复修改 `project.pbxproj`。

## [Unreleased] - 2026-05-17

### 修复
- 修复健康看板 HealthKit 权限问题，避免授权异常导致数据无法展示。
### 优化
- 圈子业务逻辑与用户体验优化，提升浏览顺畅度与交互一致性。

## [Unreleased] - 2026-05-16

### 新增
🥔 土豆圈（Beta）来了

这次更新加入了全新的「圈子」功能。

现在可以在土豆圈里：

* 分享自己的土豆片作品
* 将圈子内容传输到土豆片设备查看

这是圈子功能的第一个公开版本，目前仍在持续开发中，功能和体验还不算完善，可能会存在：

* 一些 Bug 或闪退问题
* 部分 UI 仍在调整
* 功能暂未全部开放
* 服务稳定性仍在优化

如果你在使用过程中遇到问题，欢迎反馈。


## [Unreleased] - 2026-05-13

### 优化
- 增强图像传输 `原始算法 4` 的真机显示效果：传输前提升饱和度、对比度并压暗中间调，使画面更接近小程序的鲜艳、深色表现。
- `原始算法 5` 同步使用相同增强，`原始算法 2 / 3` 使用较轻增强，`原始算法 1` 黑白逻辑保持不变。
- 将 6 色调色板中的绿色调整为纯绿，提升颜色识别和显示鲜艳度。
- 修复天气看板 AQI 三位数、污染等级徽标和污染物说明在固定模板内被裁切的问题，空间不足时优先省略“空气质量”前缀。
